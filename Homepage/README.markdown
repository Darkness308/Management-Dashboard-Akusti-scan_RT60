# AcoustiScan Homepage Demo

Eine minimale, interaktive Demo für die AcoustiScan-Homepage, die Raumakustik visualisiert. Entwickelt für Präsentationszwecke, mit Fokus auf visuelle Effekte und Interaktivität. **Ohne Tailwind CDN**, für produktionsreife Nutzung.

## Inhalte
- **Dateien**:
  - `index.html`: Hauptseite mit RT60-Rechner, 3D-Raum und Waveform-Animation.
  - `output.css`: Kompilierte Tailwind CSS-Datei.
  - `tailwind.config.js`: Tailwind-Konfiguration.
  - `package.json`: Node.js-Abhängigkeiten.
- **Technologien**:
  - HTML5, Tailwind CSS (lokal kompiliert) für Design.
  - Three.js (CDN) für 3D-Raum.
  - Chart.js (CDN) für RT60-Grafiken.
  - Web Audio API für Klang-Feedback.
